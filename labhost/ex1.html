BASIC HTML TAGS

THE BASIC HTML TAGS ARE:

TAGS DESCRIPTION
<!DOCTYPE> Defines the document type
<html> Defines an html document
<head> Contains metadata/information for the document
<body> Defines the document body

An unordered HTML list

  • Unordered list is defined by <ul> tag
  • An unordered list typically is a bulleted list of items

An ordered HTML list

  1. Ordered list is defined by <ol> tag
  2. The list items will be marked with numbers by default
